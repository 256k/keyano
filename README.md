## keyano
---
play a qwerty keyboard as a midi keyboard over virtual midi port.

### details:
2 rows of notes, each starting with C 


```
   2 3   5 6 7   9 0   =  -- sharps
  q w e r t y u i o p [ 
   s d   g h j   ;  | |
  z x c v b n m , . | |
  | | | | | | | | | | |
[ C D E F G A B C D E F ]
```
 
### controls:
UP/DOWN: change octave
LEFT/RIGHT: change velocity by jumps of 10
